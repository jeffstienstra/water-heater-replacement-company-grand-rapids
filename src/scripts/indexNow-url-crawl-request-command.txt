Run this command in the terminal to index new urls when pages are created or updated.
Use the generated sitemap for complete list of urls, then paste them in the urlList array.
The key is used to verify the request is coming from the website owner.
The keyLocation is the url to the key file

***** Key in this file MUST match key in the /public/9b143c67a55a33bce47d65885ebb51ba.txt file. *****

curl -v -X POST "https://api.indexnow.org/IndexNow" \
	-H "Content-Type: application/json; charset=utf-8" \
	-d '{
		"host": "waterheaterreplacementcompany.com",
		"key": "9b143c67a55a33bce47d65885ebb51ba",
		"keyLocation": "https://waterheaterreplacementcompany.com/9b143c67a55a33bce47d65885ebb51ba.txt",
		"urlList": [
			"https://waterheaterreplacementcompany.com/reviews/",
			"https://waterheaterreplacementcompany.com/booking/"
		]
	}'
